1. Просмотрите код программы в файле "Sensor_LINE.ino", постарайтесь понять каждую строчку кода
2. Зайдите в wokwi.com и создайте новый файл, соберите схему из светодиода и датчика освещённости(при подключении
датчика к плате используйте его цифровой вывод - DO).
3. Напишите код программы который зажжёт светодиод по цифровому сигналу от датчика освещенности.

Подсказка:
1.При написании кода программы используйте:
- создание переменных для хранения номеров выводов
- в setup() настройте выводы с помощью команды pinMode(), INPUT - датчики, OUTPUT - исполнительные устройства
- в loop() напишите основную логику програмы (получение данных с датчика, проверка этих данных, если получаем
с датчика 0 - включить светодиод, если 1 - выключить светодиод)
